# Ports
- Home: 3000
- Footer: 3001
- Banner-Count: 3002
- Banner: 3003
- Banner-Button: 3004
- Store: 3005

# Setup

Inside each folder (home, footer, banner, etc...) run yarn install & yarn start.
After this, you should have all the micro-frontends running in each port.
